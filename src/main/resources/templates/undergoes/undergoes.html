Undergoes Management

Browse and manage procedure records performed for patients during stays

Read Operations
GETAll Undergoes Records
/Undergoes/all
Retrieve every undergoes record with patient, procedure, stay, physician, and assisting nurse details.
→
GETGet by Composite ID
/Undergoes/{patient}/{procedure}/{stay}/{date}
Look up a specific record using Patient ID, Procedure ID, Stay ID, and DateUndergoes.
→
GETGet by Patient
/Undergoes/patient/{patientId}
Retrieve all procedures undergone by a particular patient.
→
GETGet by Procedure
/Undergoes/procedure/{procedureId}
See all patients and stays associated with one procedure.
→
GETGet by Stay
/Undergoes/stay/{stayId}
Retrieve all procedures recorded against a particular stay.
→
GETGet by Date Range
/Undergoes/date-range?start=&end=
Filter undergoes records by a start and end timestamp.
→
Write Operations
POSTCreate Undergoes Record
/Undergoes/create
Add a new undergoes record with its composite key and physician or nurse references.
→
PUTUpdate Undergoes Record
/Undergoes/update/{patient}/{procedure}/{stay}/{date}
Modify physician or assisting nurse values for an existing record.
→
